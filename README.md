# Gen Chart

An open-source chart generator web app. Input data via a spreadsheet grid or paste CSV/JSON, then generate bar, line, and pie charts.

## Features

- Editable spreadsheet grid for data input
- CSV/JSON paste support
- Bar, Line, and Pie chart types
- Live chart preview
- Chart options: title, axis titles, legend, data labels, units, color palettes
- Export as PNG with configurable padding
- Copy data as CSV

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- Vite + React + TypeScript
- Chart.js + react-chartjs-2
- react-datasheet-grid
- Tailwind CSS
